# 移动端的一些适配

## input ios 底部变搜索

- 放在 form 标签内。
- 在 form 标签内加上 action="javascript:return true"
- type="search"

```html
<form action="javascript:return true">
  <input type="search" placeholder="请输入" />
</form>
```

## input ios 底部变前往

- 放在 form 标签内。
- 在 form 标签内加上 action="javascript:return true"
- type="text"

```html
<form action="javascript:return true">
  <input type="text" placeholder="请输入" />
</form>
```
